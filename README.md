# Question Banks Cleaner Plugin for Moodle

A comprehensive Moodle plugin for managing and cleaning up questions in the question bank.

## Features

- View and manage used questions
- View and manage unused questions
- Identify and delete duplicate questions
- Clean up orphaned answers
- Batch deletion with progress tracking
- Course-based filtering
- Pagination support

## Installation

1. Copy the `questioncleaner` folder to `moodle/local/`
2. Visit Site administration > Notifications to complete the installation
3. Grant necessary permissions to users who should access this plugin

## Requirements

- Moodle 4.0 or higher
- PHP 8.2 or higher

## Usage

Access the plugin through:
**Site administration > Reports > Question Cleaner**
